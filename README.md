# Rust OS Project

Rustで作成している自作OSの学習用プロジェクトです。
書籍 『[［作って学ぶ］OSのしくみⅠ](https://gihyo.jp/book/2025/978-4-297-14859-1)』 に基づいて実装しています。

## 参考文献 (Reference)
- **Book:** [［作って学ぶ］OSのしくみⅠ](https://gihyo.jp/book/2025/978-4-297-14859-1) (ISBN: 978-4-297-14859-1 / 978-4-297-14860-7)
- **Author:** [@hikalium](https://github.com/hikalium)
- **Original Repository:** [hikalium/wasabi](https://github.com/hikalium/wasabi)

## 開発環境 (Environment)
以下の環境で開発・動作確認を行っています。

- **Host OS:** Windows 11 (WSL2)
- **Guest OS:** Ubuntu 24.04 LTS
- **Language:** Rust (Nightly channel)
- **Emulator:** QEMU
